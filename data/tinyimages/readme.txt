Download file http://horatio.cs.nyu.edu/mit/tiny/data/tiny_images.bin into
this directory.

tiny_index.pkl is an index for finding image ranges with given label from 
the binary file. It is generated from the original "tiny_metadata.bin" file
on the download page.

See http://horatio.cs.nyu.edu/mit/tiny/data/index.html for more information.
